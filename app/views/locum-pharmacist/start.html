About this prototype

This prototype explores how locum pharmacists (those with the FFFFF user role) can access the redesigned SCRa at the dispensing site from which they are working.

Prototype hypotheses

The redesigned SCRa may be used by locum pharmacists with the user role FFFFF. These pharmacists may work from a number of dispensing sites so we will allow them to select the site they are working from when logging in.

To provide this functionality, we will introduce an autocomplete search function to allow users to perform this selection. This is preferred to the existing method of submitting a search and then browsing the results.

If we allow users to search for a dispensing pharmacy site using autocomplete

Then they will be able to quickly select the site they are working from

Because there will be no need to submit a search and wait for the results to load.

If we show users the sites using autocomplete as they search

Then they will be sure that the site they are looking for exists

Because there will be no need to search through a large list of potential results
