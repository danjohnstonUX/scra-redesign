About this prototype

This prototype explores the creation of a screen which allows users to select the relevant role and organisation upon login, as well as change their role or organisation at any point thereafter.

There are a number of different scenarios which a user may be involved in when trying to access the service. These scenarios can be viewed using the links in the Prototypes columns.
